# Keybindings

## Quick Reference

```
╔════════════════════════════════════════════════╗
║           KEYBINDING QUICK REFERENCE           ║
╠════════════════════════════════════════════════╣
║                                                ║
║  NAVIGATION (hjkl = ←↓↑→)                      ║
║  Alt + h/j/k/l           Focus window          ║
║  Alt + Shift + h/j/k/l   Swap window           ║
║  Alt + Ctrl + h/j/k/l    Move window           ║
║  Alt + Shift + hjkl      Resize window         ║
║                                                ║
║  WINDOW STATES                                 ║
║  Alt + F                 Fullscreen            ║
║  Alt + Shift + F         Toggle float          ║
║  Alt + Shift + M         Fullscreen            ║
║  Alt + Shift + Return    Swap with master      ║
║  Alt + Q                 Kill window           ║
║                                                ║
║  MOUSE                                         ║
║  Alt + Left Click        Move window           ║
║  Alt + Right Click       Resize window         ║
║                                                ║
║  LAUNCHERS                                     ║
║  Cmd/Alt + Return        Terminal              ║
║                                                ║
║  WORKSPACES (0 = workspace 10)                 ║
║  Alt + [1-9,0]           Switch workspace      ║
║  Alt + Shift + [1-9,0]   Move to workspace     ║
║  Ctrl + ←/→              Cycle workspaces      ║
║  Ctrl + Shift + ←/→      Move + cycle          ║
║                                                ║
╚════════════════════════════════════════════════╝
```
